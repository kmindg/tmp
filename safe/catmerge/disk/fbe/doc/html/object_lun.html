LUN Object

The LUN object is...

  • Fast-bind Zeroing.
  • Non-destructive Bind
  • User Zeroing
  • LUN Shrink
  • LUN ownership
  • LUN power-save management
  • LUN change notification

LUN Control Path

The LUN control path performs the following operations:

  • Non-Destructive Bind
  • User Verify
  • User Zeroing

LUN Non-paged Metadata

  • io_counter — Number of non-cached io's that is outstanding.
  • lu_dirty — A flag to represent if the lun is dirty or not. False if io counter is 0.

    • LUN Paged Metadata

    Currently the LUN object does not use its paged metadata.
